Profiel

Over mij.

Mijn naam is Volken Koelewijn, ik ben geboren op 13 September 1999.
Ik woon in Uithoorn en ik volg momenteel een opleiding voor Game Developer op het Mediacollege Amsterdam.

Mijn hobby's zijn gamen, tekenen, over games praten en het verzamelen van games. Verder ben ik vrolijk en heb ik een goed gevoel voor humor.

Mijn talenten

Ik ben bijzonder creatief en kan mijn ideeën duidelijk uitleggen, daarnaast kan ik hiermee anderen ook motiveren en inspireren.
Ook ben ik leiding gevend ingesteld en kan ik goede planningen maken, Aansluitend kan ik ook goed comminuceren en team relaties sterk houden.
Verder ben ik erg vertrouwd over mijn kennis in Game Design en Game feel.

Wat wil ik leren

Momenteel werk ik vooral met C# in Unity.

Echter wil ik beter weten hoe ik C# kan gebruiken. En wil ik C++ op pakken.
Maar voornamelijk wil ik mijn scripting in algemeen verbeteren


Ook wil ik graag mijn kennis over game design en game feel uitbreiden.

Als laatste wil ik ook meer leren over de zakelijken aspecten van een game bedrijf runnen.

Mijn Doel!

Mijn doel is dat ik uiteindelijk een eigen bedrijf kan beginnen en mijn eigen opdrachtgever en Game Designer kan zijn,
Zodat ik mijn eigen ideeën en visie om kan zetten in games die ik maak.
