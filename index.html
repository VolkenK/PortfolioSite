Profiel

Over mij.

Mijn naam is Volken Koelewijn,
ik ben geboren op 13 September 1999. Ik woon in Uithoorn en ik volg momenteel een opleiding voor Game Developer op het Mediacollege Amsterdam.

Ik ben vrolijk en ik heb een goed gevoel voor humor. Ik speel en verzamel video games en praat er ook veel over. Verder teken ik ook graag. Inspiratie voor game concepten haal ik vaak uit vergezochte of onbenullige dingen die mij aan het denken zetten; als ik dit soort inspiratie heb werk ik ze uit door ze op papier te tekenen. Ik programmeer in unity.

Mijn Doel!

Mijn doel is dat ik instaat ben om mijn game concepten te kunnen verwezenlijken als werk.
Game concepten uitdenken en op papier zetten.
Een demo van die game programmeren
Het game concept verder uitwerken
Een team leiden om die game te maken
